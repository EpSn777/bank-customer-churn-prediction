Bank Customer Churn Prediction

This project explores a customer churn dataset from a banking institution and applies machine learning techniques to predict whether a customer is likely to leave the bank.

The analysis includes data cleaning, exploratory data analysis (EDA), feature preprocessing, and the implementation of Decision Tree and Logistic Regression models.

Dataset:
https://www.kaggle.com/datasets/shrutimechlearn/churn-modelling

Tools used:

Python
Pandas
Scikit-Learn
Matplotlib
Seaborn
Jupyter Notebook
